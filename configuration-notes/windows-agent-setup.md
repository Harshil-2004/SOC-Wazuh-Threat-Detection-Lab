# Windows Agent Setup

## Objective
Install and configure the Wazuh Agent on a Windows 11 endpoint.

## Steps Performed

1. Opened Wazuh Dashboard.
2. Navigated to Deploy New Agent.
3. Selected Windows as the operating system.
4. Entered the Wazuh Manager IP address.
5. Generated the MSI installation command.
6. Executed the command using PowerShell as Administrator.
7. Started the Wazuh Agent service.
8. Verified the agent status in the dashboard.
