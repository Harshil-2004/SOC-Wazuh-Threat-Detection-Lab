# Wazuh Manager Setup

## Objective
Deploy the Wazuh Manager as the central SIEM server for log collection, alert analysis, and dashboarding.

## Steps Performed

1. Downloaded the Wazuh OVA image.
2. Imported the OVA into VirtualBox.
3. Configured VM resources.
4. Started the Wazuh Manager virtual machine.
5. Accessed the Wazuh Dashboard using the manager IP address.
6. Verified that the Wazuh services were running.

## Configuration Notes

- Deployment method: Wazuh OVA
- Virtualization platform: VirtualBox
- Dashboard access: Browser-based HTTPS access
- Main purpose: Central log collection and alert analysis

## Outcome

The Wazuh Manager was successfully deployed and accessed through the Wazuh Dashboard.
